## [0x00] What it does

The engine is the [`spider`](https://github.com/spider-rs/spider) crawler ‚ÄĒ
this package does not reimplement crawling, it vendors spider and exposes it
to stryke, then layers HTML extraction on top. Two halves:

- **Network** (`fetch`, `crawl`, `links`, `sitemap`) drive spider through a
  single embedded tokio runtime owned by the cdylib.
- **Pure** (`extract`, `extract_table`, `extract_links`, `extract_attrs`,
  `extract_text`, `extract_meta`, `extract_images`, `extract_feeds`,
  `structured`, `select`, `absolutize`) run html5ever-backed CSS selection over
  an HTML string with no network ‚ÄĒ the unit-tested core, usable on any HTML you
  already have.

```stryke
use Scrape
val $page = Scrape::fetch "https://example.com"
val $rec  = Scrape::extract $page->{html}, { title => "h1", price => ".price" }
```

## [0x02] Quick start

```stryke
use Scrape

# 1. Fetch one page.
val $page = Scrape::fetch "https://example.com"
p "$page->{status} ‚ÄĒ #{len $page->{html}} bytes"

# 2. Extract fields. A selector ending in ` @attr` pulls that attribute,
#    otherwise the element text. `all => 1` returns every match.
val $rec = Scrape::extract $page->{html}, {
    title => "h1",
    links => "a @href",
}, all => 1

# 3. Tables ‚Üí header-keyed records.
val $rows = Scrape::extract_table $page->{html}
for val $r (@$rows) { p $r }

# 4. Structured metadata.
val $meta = Scrape::structured $page->{html}
p $meta->{opengraph}{title}

# 5. Crawl a site ‚ÄĒ robots respected, bounded, throttled.
val $res = Scrape::crawl "https://example.com",
    limit => 50, depth => 3, delay => 200, concurrency => 4
p "crawled $res->{count} pages"

# 6. Just the link graph (no bodies).
val $map = Scrape::links "https://example.com", limit => 200

# 7. Sitemap-driven crawl.
val $sm = Scrape::sitemap "https://example.com"
```

## [0x03] Crawl options

Every network fn takes the URL first, then `%opts`:

```
limit          ‚Üí max pages to fetch
depth           ‚Üí max link depth from the seed
subdomains      ‚Üí 1 to follow subdomains
respect_robots  ‚Üí robots.txt compliance (default 1)
delay           ‚Üí ms between requests (politeness throttle)
concurrency     ‚Üí max in-flight requests
timeout_ms      ‚Üí per-request timeout
blacklist       ‚Üí [ url substrings / patterns to skip ]
proxies         ‚Üí [ proxy urls ]
user_agent      ‚Üí override the default stryke-scrape/<ver> UA
include_html    ‚Üí crawl/sitemap: 0 to omit page bodies (URL map only)
```

## [0x04] API reference

```stryke
# Network (spider engine)
Scrape::fetch    $url, %opts ‚Üí { url, status, html }
Scrape::crawl    $url, %opts ‚Üí { count, pages => [ { url, status, html? } ] }
Scrape::links    $url, %opts ‚Üí { count, links => [ url, ... ] }
Scrape::sitemap  $url, %opts ‚Üí { count, pages => [ ... ] }

# Pure extraction (no network)
Scrape::extract         $html, \%fields, %opts ‚Üí \%record   # opts: all
Scrape::extract_table   $html, %opts ‚Üí [ \%row, ... ]       # opts: selector
Scrape::extract_links   $html, %opts ‚Üí [ { text, href }, ... ]   # opts: base
Scrape::extract_attrs   $html, $selector, $attr ‚Üí [ value, ... ]
Scrape::extract_text    $html, %opts ‚Üí $string              # opts: selector (default body)
Scrape::extract_meta    $html ‚Üí { title, description, canonical, ... }
Scrape::extract_images  $html, %opts ‚Üí [ { src, alt }, ... ]     # opts: base
Scrape::extract_feeds   $html, %opts ‚Üí [ { title, href, type }, ... ]   # opts: base
Scrape::structured      $html ‚Üí { jsonld => [...], opengraph => {...}, twitter => {...} }
Scrape::select          $html, $selector, %opts ‚Üí [ html, ... ]   # opts: inner
Scrape::absolutize      $base, $href ‚Üí $url

# URL / query-string helpers (pure)
Scrape::url_encode      $value ‚Üí $encoded                    # RFC 3986 percent-encode
Scrape::url_decode      $value, %opts ‚Üí $decoded             # opts: plus_as_space
Scrape::parse_query     $query ‚Üí { key => value, ... }
Scrape::build_query     \%params ‚Üí $query_string
Scrape::url_parse       $url ‚Üí { scheme, host, port, path, query, fragment, username, password }

Scrape::version ‚Üí $semver
```

Selector specs in `extract` support a trailing ` @attr`: `"a.nav @href"`
extracts the `href` attribute, a bare `"h1"` extracts text. `extract_links`
resolves relative hrefs when given a `base`.

## [0x05] FFI layer

Each `Scrape::*` wrapper builds a JSON args dict and calls a sibling
`scrape__*` symbol resolved out of `libstryke_scrape.{dylib,so}`. The cdylib
is dlopened in-process on first `use Scrape` (via stryke's
`pkg::commands::try_load_ffi_for` resolver hook). Network calls run on one
embedded multi-thread tokio runtime created lazily and reused for the life of
the process. Wire shape:

* Single record ‚Üí `{"value": ‚Ä¶}` / `{"rows": ‚Ä¶}` / `{"links": ‚Ä¶}`
* Crawl result ‚Üí `{"count": N, "pages": [...]}`
* Errors ‚Üí `{"error": "<msg>"}` ‚ÄĒ the wrapper `die`s with it

## [0x06] Politeness

`crawl`, `links`, and `sitemap` default `respect_robots` to true and send an
identifying `stryke-scrape/<version>` User-Agent. `delay` and `concurrency`
throttle a crawl. Respecting a site's terms of use, rate limits, and robots
directives is the caller's responsibility ‚ÄĒ this package provides the
mechanisms and sets the courteous defaults.
